# @nextdevx/whats-new

## 0.2.0

### Minor Changes

- Initial public release of @nextdevx packages.

  Includes:
  - @nextdevx/core: Auth adapters (Supabase, Clerk, NextAuth), ElementPicker, config provider
  - @nextdevx/devtools: DevTools component, DevLoginPage, element copy dialog
  - @nextdevx/feedback: Feedback system with screenshots and issue tracker integration
  - @nextdevx/theme: Theme provider with light/dark/system modes
  - @nextdevx/whats-new: Changelog component with voting
  - @nextdevx/audit: Audit logging with sensitive field sanitization
  - @nextdevx/cli: CLI for package setup and migrations

- Updated dependencies
